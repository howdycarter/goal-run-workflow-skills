# Viral Launch Plan

## Positioning

Frame `goal-run` as an operating loop for Codex agents, not as "four prompt files."

Core claim:

> Prompts are not enough. Agents need a durable loop for defining, delegating, and proving work.

## Why It Is Shareable

- It names a pain many agent users feel: impressive starts, weak finishes.
- It gives the pain a compact model: `Goal Define -> Goal Ultra -> Swarm -> Run + Verify`.
- It includes a repo people can copy immediately.
- It has a clear visual that explains the architecture in one glance.

## Recommended Launch Package

Post order:

1. Main X post with `goal-run-workflow-teaser.mp4`.
2. Reply with the GitHub link.
3. Reply with the Excalidraw diagram.
4. Reply with the install command.

Reason:

Putting the repo link in the first reply keeps the main post focused on the idea and the media.
